Formulario

GET

Por padrão, todo formulario tem o metodo get para o envio dos formularios

Usa-se get quando os dados não forem sensiveis

get so aceita 3 mil bytes o que equivale a 3mil caracteres

POST

Aceita mais do que 3 mil bytes, é recomendado se tiver dados mais sensiveis

Em teoria é um metodo um pouco mais lento
