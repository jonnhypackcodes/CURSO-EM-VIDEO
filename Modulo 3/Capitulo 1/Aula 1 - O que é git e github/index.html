Diferenciando GIT e GITHUB

Repositorio local e remoto

Repositorio é um local onde oce guarda coisas.

Git

É um gerenciador de versões que faz um backup de todas as versoes de seu projeto dentro de seu computador. Geralmente é chamado de Repositorio local

Github

O github faz o backup de tudo o que o git fez backup localmente. Geralmente chamado de repositorio remoto.

Github permite que outras pessoas também mexam no seu projeto

Enviar um arquivo para o repositorio remoto é chamado de PUSH
