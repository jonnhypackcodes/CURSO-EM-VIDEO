Representação de dados

No computador os dados são representados por 0 e 1. Isso significa um = ligado e 0 = desligado, chamados de bit ou numero binario.

8 bits representa 1 byte

1024 bytes é 1 kb

1024 kb = 1mb

1024mb = 1gb

Como nos conectamos com a internet

A pessoa que acessa a internet é o cliente e o site que ela acessa se chama servidor. O modem serve para modular e demodular as ondas transimitidas entre computador e internet.
