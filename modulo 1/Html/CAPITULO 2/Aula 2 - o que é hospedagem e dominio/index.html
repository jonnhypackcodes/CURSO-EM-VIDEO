Como as pessoas vão ver meu site?

Para voce conseguir mostrar pra outras pessoas o seu site, primeiramente voce precisa de um dominio. Dominio é um nome unico, não pode exister dois iguais.

Geralmente o dominio é pago, através de uma taxa anual

Hospedagem

Hospedagem é um espaço para voce armazenar arquivos do site

Pago mensalmente

URL

é um endereço que só serve pra ir pro servidor

Uma url é dividida entre, sub dominio, dominio, TLD(.com) GTLD que é uma extensão generica e CCTLD(Country)
