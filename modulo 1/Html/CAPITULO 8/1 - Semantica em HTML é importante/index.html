Semantica

Semantica é o significado das coisas

Em HTML 5, as tags se reservam a semantica, enquanto o estilo se reserva totalmente ao css

Exemplo

Ao invés de usar a tag <u> que se reserva a sublinhar alguma frase, pode-se usar a tag<adress> por exemplo, referente a endereço. ex:

Rua vinicius de moraes - 321
