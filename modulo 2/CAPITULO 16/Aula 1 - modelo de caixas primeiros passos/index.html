Modelo de caixas

Teoria das caixas

Todo site feito em HTML se constitui de caixas e hierarquia de caixas.

Uma caixa colocada dentro da outro se chama aninhamento

Todo elemento visivel dentro de um site é uma caixa

Uma caixa é constituida de margin, padding, border, outline, width e height

Tipos de caixa

Existem 2 tipos de caixa: box level e inline-level

  • BOX-LEVEL

    • Sempre ocupará a largura total da tela

  • INLINE-LEVEL

    • Continua na mesma linha e nao quebra linha
