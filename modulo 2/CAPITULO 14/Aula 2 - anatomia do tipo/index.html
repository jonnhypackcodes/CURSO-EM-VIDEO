Anatomia do tipo

Anatomia do tipo é como serão desenhadas cada uma das letras

Curiosidade: a letra X geralmente é a primeira letra desenhada quando se quer criar uma nova fonte. Também existe a altura das maiusculas que chama-se ascendente e descendente as partes das letras que vazam pra baixo. A altura total de todas as letras se mede pelo corpo, juntando as partes ascedentes com descendentes

Serifa

Serifa são linhas de marcação existentes nas partes inferiores e superiores das letras minusculas. O objetivo é que o nosso olho seja guiado

Anatomias geometricas

Anatomia geometrica representa a forma como as letras são montadas

  • Haste: são as partes que estruturam a parte vertical das letras
  • Filete: É a parte que geralmente unem as partes verticais das letras
  • Arco
  • Esporão: é a parte de baixo da letra, que serve para "segurar" a letra.
  • Vertice:
  • Terminal: é um elemento que tem uma ponta solta, não estando preso por duas hastes
  • Braço
  • Perna
  • Pé
  • Espinha
  • Barriga
  • Olho
  • Cauda
  • Glifo: Cada letra representada pela fonte tem o nome de glifo.
