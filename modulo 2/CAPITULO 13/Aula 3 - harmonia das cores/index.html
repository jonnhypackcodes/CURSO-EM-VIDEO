Harmonia das cores

O circulo cromatico

O circulo cromatico pode ser representada usando 12 cores principais

O foco do estudo do circulo cromatico é entender como harmonizar as cores

Exemplo de coisas que são mais bonitas

  • Simetria
  • Cores

    • Cores primarias

    As cores primairas são dentro do circulo cromatico sao:

    • Amarelo
    • Vermelho
    • Azul

    Cores secundarias

    As cores secundarias são representadas por:

    • Laranja
    • Violeta
    • verde

Cores terciaras

Cores terciarias sempre será a junção das cores primarias e secundarias:

  • Amarelo-esverdeado
  • Amarelo-alaranjado
  • vermelho-alaranjado

Temperatura das cores

Ao dividir o circulo cromatico no meio, podemos verificar que o lado direito possui as cores mais frias e o lado esquerdo as mais quentes

Escolhendo a paleta de cores

A paleta de cores sempre parte de uma cor principal

Geralmente uma paleta de cor te m de 3 a 5 cores no maximo, desconsiderando preto e branco

Cores complementares

Cores que mais tem contraste entre si

Dentro do circulo cromatico, criando uma linha reta entre as cores, pode-se ter as cores complementares

Cores analogas

São cores que são parecidas

Pegando as cores ao lado da sua cor principal dentro do circulo cromatico, se obtem as cores analogas

Cores triadicas

Pegando uma cor principal e pulando 3 cores do circulo cromatico 2 vezes, se obtem uma paleta triadicas

Dicas

Uma boa tecnica é pegar duas cores analogas e uma complementar
